Caso de uso: Visualização de planilha do estoque de peças (uc6)

Este caso de uso é composto pelas funcionalidades de outros casos de uso:

 - Alteração de informação de peça (uc8)
 - Retirada de peça do estoque (uc7)

O resultado de design/implementação deste caso de uso foi a geração de um componente Vue.
Este componente é exportado no arquivo visualizacao_estoque_component.js e utilizado nos casos de uso

 - Gerenciamento de Estoque (uc4)
 - Busca de peças no estoqeu (uc15)

Portanto, para ver seu funcionamento e interagir com ele, basta acessar e testar um destes casos de uso.
O componente em si não se trata de uma página, por isso não pode ser visto individualmente.
